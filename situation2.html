Contexte

Dans un contexte pédagogique, j’ai développé SkyStorm, une application web inspirée de Twitter/X, permettant la gestion d’utilisateurs et d’interactions sociales.

Objectif

Mettre en œuvre une application web complète en architecture MVC avec Laravel, incluant authentification, gestion de données relationnelles et interactions sociales.

Fonctionnalités principales

  • Inscription et connexion des utilisateurs
  • Création, modification et suppression de posts
  • Système de suivi entre utilisateurs
  • Système de likes
  • Fil d’actualité simplifié

Environnement technique

  • Langage : PHP 8
  • Framework : Laravel 12
  • Base de données : MySQL
  • ORM : Eloquent
  • Frontend : Blade / Bootstrap

Compétences mobilisées

  • Développement web en architecture MVC
  • Modélisation de base de données (MCD)
  • Gestion des relations 1-N et N-N
  • Implémentation d’un système d’authentification
  • Organisation d’un projet Git
