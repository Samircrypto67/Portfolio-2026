Contexte

Dans un contexte pédagogique pour l'épreuve E6 du BTS SIO option SLAM, j'ai développé SkyStorm, une application web inspirée de Twitter/X, permettant la gestion d'utilisateurs et d'interactions sociales. Ce projet démontre la maîtrise du framework Laravel et des bonnes pratiques de développement web moderne.

Objectif

Mettre en œuvre une application web complète en architecture MVC avec Laravel, incluant authentification, gestion de données relationnelles et interactions sociales. Le projet vise à démontrer les compétences en développement web full-stack et en gestion de projet.

Fonctionnalités principales

  • Inscription et connexion des utilisateurs avec Laravel UI
  • Création, modification et suppression de posts
  • Système de suivi entre utilisateurs (follow/unfollow)
  • Système de likes sur les posts
  • Fil d'actualité personnalisé
  • Interface responsive avec Bootstrap

Architecture et modélisation

L'application suit l'architecture MVC de Laravel avec une séparation claire des responsabilités :

  • Modèles (Models) : Gestion des entités User et Post avec Eloquent ORM
  • Vues (Views) : Templates Blade pour l'interface utilisateur
  • Contrôleurs (Controllers) : Logique métier et gestion des requêtes HTTP
  • Routes : Définition des endpoints et middlewares d'authentification

Modèle de données (MCD) :

  • Relation N-N entre Users (système de follow)
  • Relation 1-N entre Users et Posts
  • Relation N-N entre Users et Posts (système de likes)

Environnement technique

  • Langage : PHP 8.x
  • Framework : Laravel 12
  • IDE : PhpStorm
  • Base de données : MySQL / MariaDB
  • ORM : Eloquent
  • Frontend : Blade templates / Bootstrap (Laravel UI)
  • Build tool : Composer (dépendances PHP)
  • Asset compilation : Vite.js
  • Testing : PHPUnit
  • Versionning : Git

Compétences mobilisées

  • Développement web en architecture MVC avec Laravel
  • Configuration et utilisation de PhpStorm pour le développement PHP
  • Modélisation de base de données (MCD/MLD) et migration Laravel
  • Gestion des relations 1-N et N-N avec Eloquent ORM
  • Implémentation d'un système d'authentification complet
  • Développement d'API RESTful avec Laravel
  • Utilisation de Blade pour les templates dynamiques
  • Intégration de Bootstrap pour l'interface utilisateur
  • Organisation et gestion d'un projet Git professionnel
  • Tests unitaires et fonctionnels avec PHPUnit

Résultats et livrables

SkyStorm a été développé avec succès et démontre une application web fonctionnelle avec toutes les fonctionnalités sociales demandées. Le code source est organisé selon les standards Laravel, documenté et versionné sur Git. L'application est déployable et prête pour une utilisation en production.
